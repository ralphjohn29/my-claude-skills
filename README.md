# 🚀 My Claude Code Skills

A collection of custom skills for [Claude Code](https://claude.com/claude-code) - my AI coding assistant configuration.

## 📦 Skills Included

- **ui-ux-pro-max** - Advanced UI/UX design with 50+ styles, palettes, and font pairings
- **ui-ux-designer** - UI/UX design copilot for wireframes, prototypes, and design systems
- **frontend-design** - Production-grade frontend interfaces with high design quality
- **frontend-alignment-doctor** - Diagnose and fix layout/alignment issues
- **frontend-ux-qa-pro** - Professional UI/UX quality assurance with Playwright
- **senior-frontend** - React, NextJS, TypeScript, Tailwind CSS best practices

### Backend Development

## 📥 Installation

### Method 1: Clone to your Claude skills directory
```bash
# Clone this repo
git clone https://github.com/ralphjohn29/my-claude-skills.git

# Copy skills to your Claude directory
cp -r my-claude-skills/* ~/.claude/skills/
```

### Method 2: Download specific skills
```bash
# Just download what you need
curl -L https://api.github.com/repos/ralphjohn29/my-claude-skills/contents/senior-frontend
```

## 🔄 Updating

```bash
cd ~/.claude/skills
git pull origin main
```

## 📝 Skill Structure

└── references/        # Additional reference documentation
